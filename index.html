1. 나라장터 수집공사+용역+물품 병렬 API (최대 5,000건)
2. AI 점수 산정낙찰방법·마감일 기반 6개 항목 자동 평가
3. 등급 분류A(70+) / B(50~69) / C(49-) 자동 구분
4. 수집 후 자동반영대시보드·오늘의영업·파이프라인 동시 업데이트
5. 파이프라인 관리연락전→연락중→미팅→계약 칸반 추적

💡 효과적인 활용 팁

전체수집보다 최근 30일 + 300건 조합이 진행중 공고 비율이 높아 실질 A등급이 더 많이 나옵니다
